## 🎨 Customize Your Site

### Add Real Content

1. **Sermons** → `components/sections/Sermons.tsx`
2. **Events** → `components/sections/Events.tsx`
3. **Testimonials** → `components/sections/Testimonials.tsx`

### Change Colors

Edit `tailwind.config.ts` - Change primary color from purple to your choice

### Add Images

Place images in `/public` folder and update component paths

3. Import repository

## 🆘 Need Help?

### Common Issues

**Port 3000 already in use?**
```bash
# Kill the process
npx kill-port 3000
# Then restart
npm run dev
```

**Dependencies not installed?**
```bash
npm install
```

**Database errors?**
```bash
npx prisma generate
npx prisma migrate dev
```
